# Employee-Performance-Dashboard-Power-BI-
This project is a Power BI dashboard built to analyze employee KPIs across departments and regions. It includes metrics such as attendance, salary distribution, performance ratings, project handling, and target achievements.


# Key Features:
1. KPI Cards: Avg. Attendance, Avg. Salary, Avg. Rating, Projects Handled
2. Department-wise & Region-wise performance analysis
3. Data cleaning & modeling using Power Query
4. Visuals: Bar, line, donut chart, pie chart, funnel, slicers
5. Insights for HR decision making

# Skills Demonstrated: Power BI, Power Query, Data Cleaning, Visualization, KPI Analysis, Storytelling
